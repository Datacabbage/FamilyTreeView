# FamilyTreeView
### 家谱树绘制Demo，暂时还在开发中，主要使用自定义ViewGroup，和使用canvas进行划线，现实现了自己、配偶、兄弟姐妹、父母、祖父母、外祖父母、子女以及孙子的绘制，加入了touch事件可以移动。代码可能写得相对比较死，基本只能使用于家谱展示
